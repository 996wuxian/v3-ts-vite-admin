# v3-ts-vite-admin
这是一个使用了vite + vue3 + ts + element-plus的前端模板，里面包含了动态路由，角色管理，菜单管理等权限配置

后端请下载仓库下代码： 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
